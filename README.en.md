# file-uploader

#### Description
以vue-uploader及若依框架为基础搭建的文件上传系统， 支持多文件上传、大文件上传，可暂停文件上传，支持快传，有上传队列管理，支持分块上传，支持进度、预估剩余时间、重试、重传等。且支持了在系统宕机后尽可能的保证文件上传进度能继续。
